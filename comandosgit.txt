COMANDOS GITHUB

- git init
- git add . 
- git status 
- git commit -m ""
- git remote add origin URL do meu github
- git branch -m main
- git push -u origin main


or create a new repository on the command line
echo "# github" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/rosanahandrade/github.git
git push -u origin main


or push an existing repository from the command line
git remote add origin https://github.com/rosanahandrade/github.git
git branch -M main
git push -u origin main
